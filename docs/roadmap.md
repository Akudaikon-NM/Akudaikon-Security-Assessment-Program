# Roadmap Highlights

## v1 Focus
- **Requirement-Anchored Scoring**: Pass/Partial/Fail scoring per question based on enforceable requirements.
- **Evidence Index Export**: Generate board-ready and examiner-ready reports.
- **Optional AI Assist**: Document summarization and gap detection with mandatory human approval gates.

## Future Versions
- Enhanced automation for evidence processing.
- Integration with external compliance databases.
- Advanced reporting and analytics.

See the product roadmap section in the repo documentation for more details.
